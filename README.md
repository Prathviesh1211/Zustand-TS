## Setup & Run
1. Clone the repo:
   ```bash
   git clone https://github.com/Prathviesh1211/Zustand-TS.git
   cd zustand-todo
   ```
   
2.Install dependencies:
```bash
npm install
```

3.Start development server:
```bash
npm run dev
```
